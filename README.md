# 📊 Amazon Sales Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing Amazon sales data using Python and Power BI.

The objective is to identify sales trends, profit patterns, top-performing products, and shipping performance through exploratory data analysis (EDA) and an interactive dashboard.

---

## 🛠️ Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI
- Jupyter Notebook

---

## 📂 Dataset Information

The dataset contains information about:

- Order Details
- Product Categories
- Sales
- Profit
- Quantity
- Shipping Days
- Region Information

---

## 📈 Exploratory Data Analysis (EDA)

The following analyses were performed:

- Data Cleaning
- Missing Value Check
- Sales Analysis
- Profit Analysis
- Category-wise Performance
- Monthly Sales Trends
- Shipping Days Analysis
- Top Product Analysis

---

## 📊 Power BI Dashboard Features

- KPI Cards
  - Total Sales
  - Total Profit
  - Total Quantity
  - Average Shipping Days
  - Profit Margin

- Interactive Filters
  - Year Filter
  - Month Filter

- Visualizations
  - Monthly Profit Trend
  - Top 10 Products
  - Profit vs Sales by Category
  - Quantity by Shipping Days

---

## 🔍 Key Insights

- Identified top-selling products.
- Analyzed monthly profit trends.
- Compared profit and sales across categories.
- Evaluated shipping performance.
- Built an interactive business dashboard.
